# 🧠 AI Security System

Адаптивная система защиты сервера с элементами искусственного интеллекта.

## 🎯 Возможности

### **Интеллектуальный анализ**
- ✅ Распознавание паттернов атак (Brute Force, Credential Stuffing, Distributed)
- ✅ Оценка репутации IP в реальном времени
- ✅ Географический анализ рисков
- ✅ Временной анализ (ночные атаки подозрительнее)

### **Адаптивное реагирование**
| Уровень | Действие | Длительность |
|---------|----------|--------------|
| 🟢 LOW | Мониторинг | - |
| 🟡 MEDIUM | Временная блокировка | 10 минут |
| 🟠 HIGH | Длительная блокировка + подсеть | 24 часа |
| 🔴 CRITICAL | Перманентная блокировка | Навсегда |

### **Самообучение**
- Анализ эффективности мер защиты
- Корректировка порогов срабатывания
- Прогнозирование следующих целей атаки
- База знаний об атаках

## ⚙️ Настройка

### Изменение порогов срабатывания

Откройте `ai_config.conf`:

```bash
sudo nano /home/larson/security-monitor/ai_config.conf
```

**Основные параметры:**

```bash
# Пороги (0-100)
THRESHOLD_LOW=20        # Низкий уровень
THRESHOLD_MEDIUM=40     # Средний уровень
THRESHOLD_HIGH=60       # Высокий уровень
THRESHOLD_CRITICAL=80   # Критический уровень

# Веса факторов
WEIGHT_FAILED_LOGIN=10      # Неудачный логин
WEIGHT_ROOT_ATTEMPT=20      # Попытка root
WEIGHT_BRUTE_FORCE=30       # Перебор
WEIGHT_GEO_RISK=10          # Гео-риск

# Риски стран
GEO_RISK_CHINA=30
GEO_RISK_RUSSIA=25
GEO_RISK_NORTH_KOREA=40
```

## 🧠 Как работает ИИ

### 1. Сбор данных
```
Неудачные логины → Геолокация → Время → История IP
```

### 2. Оценка угрозы
```
Репутация IP + История атак + Гео-риск + Время = Уровень угрозы
```

### 3. Распознавание паттерна
```
SSH Brute Force
Root Targeting
Distributed Attack
Credential Stuffing
```

### 4. Адаптивный ответ
```
LOW → Мониторинг
MEDIUM → Временная блокировка
HIGH → Длительная + подсеть
CRITICAL → Перманентная
```

### 5. Обучение
```
Результат → Анализ эффективности → Корректировка
```

## 📊 Статистика обучения

Система записывает все действия в `learning_database.dat`:

```
2026-03-09 00:30:45|117.50.70.125|LONG_BAN|success
2026-03-09 00:31:12|83.150.21.135|TEMP_BAN|success
```

Эффективность мер:
- `LONG_BAN`: 95% (блокирует повторные атаки)
- `TEMP_BAN`: 60% (некоторые возвращаются)
- `PERM_BAN`: 100% (навсегда)

## ⚠️ Важные заметки

1. **Не удаляйте файлы состояния** — там история обучения
2. **Регулярно проверяйте чёрный список** — могут быть ложные срабатывания
3. **Настройте пороги** под вашу среду
4. **Мониторьте логи** — для понимания работы ИИ

## 🆘 Troubleshooting

**Служба не запускается:**
```bash
sudo systemctl status ai-security
sudo journalctl -u ai-security -n 50
```

**Нет уведомлений:**
```bash
cat /home/larson/security-monitor/tg_config.conf
# Проверьте токен и Chat ID
```

**Слишком много блокировок:**
```bash
# Увеличьте пороги в ai_config.conf
THRESHOLD_MEDIUM=50
THRESHOLD_HIGH=70
```
